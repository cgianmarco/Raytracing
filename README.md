# Raytracing

## Overview

A high-performance raytracer implementation capable of rendering complex scenes with sophisticated geometry and materials in seconds. Developed and optimized for low-cost hardware, this project uses OpenCL for cross-platform GPU acceleration without requiring CUDA support. Successfully tested on a Chuwi PC with impressive results.
